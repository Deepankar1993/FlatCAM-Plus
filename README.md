=====================================================================

FlatCAM is a program for preparing CNC jobs for making PCBs on a CNC router.
Among other things, it can take a Gerber file generated by your favorite PCB
CAD program, and create G-Code for isolation routing.

=====================================================================

About this fork
---------------

This fork (https://github.com/Deepankar1993/FlatCAM-Plus) tracks FlatCAM Evo
Beta and adds, among other things:

- a **Laser plugin** (Menu -> Plugins -> Laser) aimed at GRBL machines fitted
  with a diode laser (e.g. CNC 3018 + laser module with air assist), driven by
  a sender such as LaserGRBL: pick a Gerber or Geometry object, choose a
  material preset (or set Power %/Speed/Beam width/Passes/Pass overlap),
  generate a laser job and export a ready-to-run `.nc` file. Gerber sources
  are traced on their centerline automatically, so PCB work needs no manual
  isolation step.
- a **GRBL_laser_air_assist** preprocessor: GRBL 1.1 laser G-code plus air
  assist control (M8 at job start, M9 at job end - wire the air pump relay to
  the controller coolant/flood output).
- automatic conversion of **legacy projects** (made with older app versions)
  when they are opened, instead of crashing.
- Windows build tooling: a PyInstaller portable build and an Inno Setup
  installer (see below).

NOTE: laser power/speed presets are conservative starting points. Materials
and laser modules vary a lot - always test on a piece of scrap first.

Works with Python 3.10 or greater (3.11 verified) and PyQt6.

**************************************************************************
1. Windows
**************************************************************************

The easy way - download from the Releases page
(https://github.com/Deepankar1993/FlatCAM-Plus/releases):

- `FlatCAM_Evo_<version>_setup.exe` (recommended): installer with Start Menu
  shortcut and uninstaller. It installs per-user, no admin rights needed.
- `FlatCAM_Evo_<version>_Windows_x64.zip`: portable build; unzip and run
  `FlatCAM_Evo\FlatCAM_Evo.exe`.

From sources:

```
git clone https://github.com/Deepankar1993/FlatCAM-Plus.git
cd FlatCAM-Plus
python -m venv venv
venv\Scripts\python -m pip install -r requirements.txt
venv\Scripts\python flatcam.py
```

Notes:
- `gdal` and `rasterio` from requirements.txt are not used by the application
  and may fail to install on Windows; it is safe to skip them.
- To build your own portable package and installer:
  `.\build_windows.ps1 -Installer` (the installer step needs Inno Setup 6:
  `winget install -e --id JRSoftware.InnoSetup`).

**************************************************************************
2. Linux
**************************************************************************

- get the sources:

```
git clone https://github.com/Deepankar1993/FlatCAM-Plus.git
cd FlatCAM-Plus
```

- make sure Python 3.10+ and pip are installed, e.g. on Ubuntu-like systems:

```
sudo apt-get install python3-pip
```

- install the dependencies and run:

```
./setup_ubuntu.sh
python3 flatcam.py
```

- alternatively install it with make:

```
# optional, if dependencies are missing
make install_dependencies

# install for the current user only
make install

# system-wide installation
sudo make install
```

**************************************************************************
3. macOS
**************************************************************************

Instructions adapted from:
https://gist.github.com/natevw/3e6fc929aff358b38c0a#gistcomment-3111878

- get the sources:

```
git clone https://github.com/Deepankar1993/FlatCAM-Plus.git
cd FlatCAM-Plus
```

- check that Homebrew is installed:

```
xcode-select --install
ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

- install dependencies:

```
brew install pyqt
brew install gdal
python3 -m ensurepip
python3 -m pip install -r requirements.txt
```

- run FlatCAM:

```
python3 flatcam.py
```
